# 🛒 End-to-end A/B testing analysis with funnel and business impact evaluation.

## 📌 Overview
Analyzed the impact of a recommendation system using an A/B testing framework. Compared control vs treatment groups to evaluate improvements in conversion and revenue.

---

## 🎯 Objective
- Measure effectiveness of recommendation system  
- Analyze user behavior across funnel  
- Evaluate business impact (conversion & revenue)  

---

## 📊 Dataset
- 5000 users  
- ~97K user activity events  

**Tables:**
- users.csv  
- experiment_assignments.csv  
- user_activity.csv  

---

## 🛠️ Tech Stack
- Python (Pandas, NumPy)  
- SQL  
- Power BI  

---

## 🔬 Approach
- Simulated user behavior data  
- Performed A/B test (control vs treatment)  
- Conducted funnel analysis:  
  View → Click → Add to Cart → Purchase  
- Built interactive dashboard for insights  

---

## 📈 Key Insights
- ~2x higher conversion rate in treatment group  
- Significant revenue uplift observed  
- Consistent improvement across devices and countries

## 🚀 Conclusion
Recommendation system significantly improves user conversion and revenue, demonstrating strong business impact.
